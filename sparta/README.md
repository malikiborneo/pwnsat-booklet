# SPARTA and Attack Flow Artifacts

This folder contains SPARTA and Attack Flow Builder artifacts used to map PwnSat / FlatSat observations, attack paths, and defensive considerations to a space-domain cybersecurity framework.

The files in this folder support thesis work related to threat-informed defense, attack-chain modeling, and satellite service disruption analysis.

---

## File Index

| File | Purpose |
|---|---|
| `Pwnsat - Assisted Analysis.afb` | Attack Flow Builder artifact for assisted analysis. |
| `Pwnsat - SPARTA TTPs (1).afb` | General SPARTA TTP mapping artifact. |
| `Pwnsat - SPARTA TTPs - IA.afb` | Initial Access-related SPARTA mapping. |
| `Pwnsat - SPARTA TTPs - RD.afb` | Reconnaissance / Discovery-related SPARTA mapping. |
| `Pwnsat - SPARTA TTPs - EX.afb` | Exploitation / Execution-related SPARTA mapping. |
| `Pwnsat - SPARTA TTPs - ExF.afb` | Exfiltration-related SPARTA mapping. |
| `Pwnsat - SPARTA TTPs - DE.afb` | Defense Evasion-related SPARTA mapping. |
| `Pwnsat - SPARTA TTPs - IMP.afb` | Impact-related SPARTA mapping. |

---

## Thesis Role

This folder supports:

- SPARTA mapping,
- attack-chain documentation,
- threat-informed defensive analysis,
- service disruption modeling,
- comparison between technical lab findings and space cybersecurity frameworks.
