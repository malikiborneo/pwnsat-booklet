# Static Assets

This folder contains images, diagrams, screenshots, and visual assets used in the PwnSat / FlatSat booklet and thesis documentation.

These assets support explanation of the lab setup, FlatSat hardware, protocol analysis, SPARTA mapping, firmware behavior, and experiment evidence.

---

## Asset Types

This folder may contain:

- FlatSat hardware photos,
- subsystem diagrams,
- Black Hat Asia lab screenshots,
- protocol analysis screenshots,
- logic analyzer screenshots,
- firmware flashing screenshots,
- architecture diagrams,
- booklet figures.

---

## Thesis Role

This folder supports:

- visual explanation of the FlatSat system,
- documentation of lab setup and hardware,
- evidence for protocol and firmware analysis,
- figures for booklet chapters,
- figures for thesis writing and presentation.
