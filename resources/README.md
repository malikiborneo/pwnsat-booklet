# Resources

This folder contains supporting lab resources for PwnSat / FlatSat analysis.

The resources in this folder are used as evidence for protocol analysis, hardware behavior observation, and experiment reproducibility.

---

## File Index

| File | Purpose |
|---|---|
| `Flatsat_v1_Initial_start.logicdata` | Raw logic analyzer capture from the initial FlatSat startup observation. |
| `Flatsat_v1_Initial_start_exported.txt` | Exported text version of the initial FlatSat startup capture. |

---

## Thesis Role

This folder supports:

- lab evidence collection,
- logic analyzer analysis,
- I2C or signal behavior review,
- protocol analysis,
- reproducibility of observations.

---

## Notes

Raw capture files should be kept unchanged.

If a capture is processed or converted, save the processed result as a separate file instead of overwriting the original.
